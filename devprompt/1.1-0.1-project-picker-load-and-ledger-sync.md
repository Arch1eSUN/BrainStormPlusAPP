## 1. 必须修复 B1 — Project picker 初始数据加载

当前问题：

`TaskListView.swift` 只有：

```swift
.task {
    await viewModel.fetchTasks()
}
```

`fetchProjects()` 只在 refreshable 里调用，导致初次打开 create sheet 时项目 picker 可能只有 `None`。

必须修：

```swift
.task {
    await viewModel.fetchTasks()
    await viewModel.fetchProjects()
}
```

或采用更稳方案：打开 create sheet 前确保 `fetchProjects()` 已完成/触发。

验收要求：

- 初始进入 Tasks 页面后，project picker 的数据源会被加载。
- refresh 仍可更新 tasks 和 projects。
- 不新增跨模块假数据。
